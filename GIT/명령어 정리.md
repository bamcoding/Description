git remote update => 깃 원격주소에 있는 브랜치 정보를 가져온다.



git remote -r => 가져온 브랜치 목록을 확인한다.

git remote -a => 로컬 브랜치 목록을 확인한다.

git checkout [원격브랜치명]


원격에 브랜치 생성
git push --set-upstream origin [branch_name]

원격에 브랜치 제거
git push origin --delete [branch_name]
